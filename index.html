PlayWithMe

PlayWithMe is a web app developed on top of Django and SQLite which connects gamers looking to play games together online or in person. It allows users to create and search for different "sessions" by game, location, and console, and then communicate with other members via an online chat. My five teammates and I poured dozens of hours into planning, prototyping, and developing this idea, and we all had a great time pitching the design in front of our class of 200 students!

Check out the GitHub repository.

Where's BabyBerk

Where's BabyBerk is a responsive webapp that allows students on the UMass campus to check at a glance where the "BabyBerk" foodtrucks are, where they will be, and what food they are serving. Where's BabyBerk uses a NodeJS backend which, using the Twitter API, obtains the most recent tweets about BabyBerk, parses them with natural language processing libraries, and logically interprets them based on their recency.

Visit Where's Babyberk, or check out the GitHub repository.

Elemental 2048

Elemental 2048 is a remix of the viral block-sliding game 2048 (written in JavaScript), with additional rules for added complexity. Players work with grass, fire, and water blocks which combine when the moving block "beats" the other. In addition, players can plan out their moves with the "next tile" indicator, and the knowledge that new tiles will always spawn on the side opposite from where they last moved. I've never beaten my game personally but I believe it might be possible with enough practice!

Try playing Elemental 2048, or check out the GitHub repository.

fastcatch

fastcatch is a fast-paced, monochromatic arcade game developed using vanilla HTML5 canvas and JavaScript, with support for accelerometer-based controls on mobile devices. In this game, you control a paddle using the arrow keys (or by tilting your phone) to try and guide a ball towards targets for points. I developed this game all the way back in high school, but it still functions well as an example of accelerometer-based user input on the web (even if you look silly while playing the game)!

Try playing fastcatch, or check out the GitHub repository.
